# Thumbnails

> Documentation > Thumbnails

Wikidot automatically generates thumbnails in various sizes for:

- every Wikidot site
- every theme on [the themes project](http://themes.wikidot.com)

The URL for site thumbnail is:

http://thumbnail.wdfiles.com/thumbnail/site/*<site_domain_name>*/*<size>*.jpg

<site_domain_name>: the main URL for site, for example: www.wikidot.com

<size>: one of 160, 80, 40, 20

The URL for theme thumbnail is:

http://thumbnail.wdfiles.com/thumbnail/theme/*<theme_name>*/*<size>*.jpg

<theme_name>: the unix name of theme

<size>: one of 500, 240, 160, 80

Examples:

- [http://thumbnail.wdfiles.com/thumbnail/site/sfugamedev.wikidot.com/160.jpg](http://thumbnail.wdfiles.com/thumbnail/site/sfugamedev.wikidot.com/160.jpg)
