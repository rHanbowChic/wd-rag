# Modules

> Documentation > Modules

Modules are built-in applications that you can place anywhere on your pages.

## Listing & Categorizing

- **[ListDrafts](/doc-modules:listdrafts-module)** — module for listing Draft Pages on site
- **[ListPages](/doc-modules:listpages-module)** — module for selecting, processing and listing pages
- **[CountPages](/doc-modules:countpages-module)** — module for counting pages
- **[ListUsers](/doc-modules:listusers-module)** — module for selecting, processing and listing users
- **[TagCloud](/doc-modules:tagcloud-module)** — creates a customizable tag cloud for tagged pages
- **[PageCalendar](/doc-modules:pagecalendar-module)** — lists number of new pages per year/month (nice for blogging)
- **[PageTree](/doc-modules:pagetree-module)** — displays a nested tree of pages connected by parenthood
- **[Backlinks](/doc-modules:backlinks-module)** — lists pages that link back to the current page
- **[WantedPages](/doc-modules:wantedpages-module)** — lists not existing pages, but linked from somewhere
- **[OrphanedPages](/doc-modules:orphanedpages-module)** — lists existing pages, but not linked from anywhere
- **[Categories](/doc-modules:categories-module)** — module for listing categories in the current site

## Users & Members

- **[Watchers](/doc-modules:watchers-module)** — list watchers of the page / category / site
- **[Members](/doc-modules:members-module)** — list members within a site
- **[Join](/doc-modules:join-module)** — allow users to join the site with one click
- **[SendInvitations](/doc-modules:sendinvitations-module)** — allow all members of a Wiki to send email invitations
- **[WhoInvited](/doc-modules:whoinvited-module)** — check who invited a particular member

## Miscellaneous

- **[CSS](/doc-modules:css-module)** — lets you add custom CSS to any page.
- **[NewPage](/doc-modules:newpage-module)** — makes creation of new pages easier.
- **[Redirect](/doc-modules:redirect-module)** — performs a "301" redirect from a page with optional URL mapping
- **[ThemePreviewer](/doc-modules:themepreviewer-module)** — allows previewing themes without altering Site configuration
- **[MailForm](/doc-modules:mailform-module)** — define a form, let others fill it and receive the forms via email
- **[PetitionAdmin](/doc-modules:petitionadmin-module)** — allows you to create online petition campaigns (like [this one](http://www.noooxml.org/petition))
- **[SiteGrid](/doc-modules:sitegrid-module)** — creates a grid of sites' thumbnails with descriptions
- **[FeaturedSite](/doc-modules:featuredsite-module)** — created clickable bigger thumbnail of site with a description

## News & Feeds

- **[Feed](/doc-modules:feed-module)** — imports RSS/Atom feeds from almost any web location and displays them in a customizable way
- **[FrontForum](/doc-modules:frontforum-module)** — uses forum categories to create news items to put on the pages and create RSS feeds

## Forum & Discussion

- **[Comments](/doc-modules:comments-module)** — put comments below page content; a cool one!
- **[RecentPosts](/doc-modules:recentposts-module)** — displays recent forum posts
- **[MiniRecentThreads](/doc-modules:minirecentthreads-module)** — displays recent forum topics
- **[MiniActiveThreads](/doc-modules:miniactivethreads-module)** — displays most active forum topics within some time period
- **[MiniRecentPosts](/doc-modules:minirecentposts-module)** — displays recent forum posts

## Page Rating

- **[Rate](/doc-modules:rate-module)** — displays a widget used to rate pages within the page body
- **[RatedPages](/doc-modules:ratedpages-module)** — displays top-rated pages

## Photos, Galleries, Files

- **[Gallery](/doc-wiki-syntax:images#gallery)** — creates gallery with nice image presentation
- **[FlickrGallery](/doc-modules:flickrgallery-module)** — pulls images from [Flickr](http://www.flickr.com)
- **[Files](/doc-modules:files-module)** — lists all files attached to a page

## Searching

- **[Search](/doc-modules:search-module)** — search one Wikidot site
- **[SearchAll](/doc-modules:searchall-module)** — search all Wikidot sites
- **[SearchUsers](/doc-modules:searchusers-module)** — search all Wikidot users

## Changes

- **[SiteChanges](/doc-modules:sitechanges-module)** — list recent changes to the site content

## Administration

- **[ManageSite](/doc-modules:managesite-module)** — site administration
- **[Clone](/doc-modules:clone-module)** — site clone tool

## Deprecated

- **[AdSenseUnit](/doc-modules:adsenseunit-module)** — insert AdSense unit into a panel (only for enabled sites).
- **[MembershipApply](/doc-modules:membershipapply-module)** — allow users to apply to the site admin(s) for membership
- **[MembershipByPassword](/doc-modules:membershipbypassword-module)** — when a password is specified people can become members of the site if they know the password
- **[Pages](/doc-modules:pages-module)** — module for listing pages within a site
- **[ChildPages](/doc-modules:childpages-module)** — lists child pages of a page
- **[PagesByTag](/doc-modules:pagesbytag-module)** — lists all pages, that are tagged with a specific tag
- **[NextPage](/doc-modules:nextpreviouspage-module)** — create links to the next page in the category.
- **[PreviousPage](/doc-modules:nextpreviouspage-module)** — create links to the previous page in the category.
- **[SimpleToDo](/doc:simpletodo-module)** — create simple "todo" lists
