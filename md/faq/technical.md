### Which web browsers are supported?

At Wikidot we use graded browser support:

| Browser release date | Support level |
| --- | --- |
| last 3 years | full support |
| 3-5 years old | partial support |
| older than 5 years | no support |

---

| Browser brand | Support level |
| --- | --- |
| Mozilla Firefox | full support |
| Google Chrome | full support |
| Apple Safari | full support |
| mobile browsers | partial support |
| other | partial support |

---

full support — everything should work

partial support — core functions work, however glitches and errors in the presentation layer and interface might occur

no support — Wikidot might work, but it comes without any guarantee

To get best Wikidot experience, please use the newest browser versions available.
