# Tags

> Documentation > Data Forms > Tags

It is not currently possible to set tags when saving the data form based on the values in the data form.

However a workaround is possible until a tag field is implemented. This workaround is described at [http://community.wikidot.com/forum/t-402555/automatically-setting-tags-for-a-page-based-on-form-input](http://community.wikidot.com/forum/t-402555/automatically-setting-tags-for-a-page-based-on-form-input)
