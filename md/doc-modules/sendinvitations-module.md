# SendInvitations Module

> Documentation > Modules > SendInvitations Module

## Description

Allows Members of a Wiki to send email invitations to their friends/coworkers to join the Wiki. This is a nice thing if you want your community to grow quickly.

For the module to work it must be enabled under Site Manager » Members » Let Users invite

## Attributes

No attributes required.

## Examples

```
Invite your friends to this Wiki!

[[module SendInvitations]]
```
