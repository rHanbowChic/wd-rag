# MembershipApply Module

> Documentation > Modules > MembershipApply Module

**This module is deprecated. Use the [Join module](/doc:join-module) instead.**

## Description

Allows registered users (these having a valid wikidot account) to apply for membership in a specific site. Applications can be reviewed by site administrators via [ManageSite](/doc:managesite-module) module under a specific tab.

Site administrators must allow applying by checking an option in the site administration module.

## Attributes

No attributes required
