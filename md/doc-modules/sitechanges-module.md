# SiteChanges Module

> Documentation > Modules > SiteChanges Module

## Description

Lists recent changes made to the site content. Supported types of changes:

- new pages
- page content changes (edit source/title)
- page rename/move
- metadata change (parent page)
- attached file actions

Changes do not include any forum activity. Use [RecentPosts](/doc:recentposts-module) module for this.

The module shows the category name for pages not in default category

## Attributes

No attributes required

[[module SiteChanges]]
