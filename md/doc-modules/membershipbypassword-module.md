# MembershipByPassword Module

> Documentation > Modules > MembershipByPassword Module

**This module is deprecated. Use the [Join module](/doc:join-module) instead.**

## Description

Allows registered users (these having a valid wikidot account) to automatically become members of the site provided they know the password.

The password itself must be configured using the [ManageSite](/doc:managesite-module) module and the password-based membership sign-in must be enabled.

## Attributes

No attributes required.

## Examples

```
[[module MembershipByPassword]]
```
