# WhoInvited Module

> Documentation > Modules > WhoInvited Module

## Description

This module will allow your users to look up how particular Members joined this Wiki. In particular it will display a chain of invitations.

## Attributes

No attributes required.

## Examples

```
Check how particular Members joined this Wiki.

[[module WhoInvited]]
```

And looks like this (this will not work very well within this Wiki because there are not many members here):

Check how particular Members joined this Wiki.

| Who invited this guy? | Type name of the user |
| --- | --- |
