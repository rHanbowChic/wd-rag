# ManageSite Module

> Documentation > Modules > ManageSite Module

**This module is deprecated.**

## Description

The ManageSite module provides you with a panel to manage your site, users, and so on. On new sites, the admin:manage page contains this module. You can put this module on any page but only logged-in site administrators are allowed to use the panel. Any other users will get a permission error.

## Attributes

This module does not take any attributes.

## Examples

Please this code on a page by itself:

```
[[module ManageSite]]
```
