# Watchers Module

> Documentation > Modules > Watchers Module

## Description

This module is used to list users watching the page. It will display all users who will receive notification about the changes on this page, so: page watchers, category watchers and site watchers.

Module Watchers also displays "watch/unwatch" options for users.

## Attributes

| attribute | required | allowed values | default | description |
| --- | --- | --- | --- | --- |
| noActions | no | "true" | none | does not show "watch/unwatch" options |

## Examples

List all users watching this page:

```
[[module Watchers]]
```
