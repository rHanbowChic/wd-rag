# Categories Module

> Documentation > Modules > Categories Module

The Categories module is a tool that displays categories and their pages within a site. This tool will eventually be replaced by [ListPages](/doc:listpages-module), which is more general purpose.

## Attributes

| attribute | required | allowed values | default | description |
| --- | --- | --- | --- | --- |
| includeHidden | no | true | false | includes "hidden" (starting with underscore) pages in the listing |

## Example

To use Categories, add this code to your page:

```
[[module Categories]]
```
