# PetitionAdmin Module

> Documentation > Modules > PetitionAdmin Module

## Description

This module allows you to set up and manage online internet petitions. You can set up several campaigns to collect signatures, download data directly into your spreadsheet application (OpenOffice Calc, Gnumeric or Excel) etc. Just awesome. One of the examples can be found here: [http://noooxml.wikidot.com/petition](http://noooxml.wikidot.com/petition)

## Attributes

None required. During the setup you will be instructed how to configure your site to run the petition campaign.

## Example

Put this piece of code on any of your (admin) pages:

```
[[module PetitionAdmin]]
```

You will be able to administer your petition from the above module provided that you have administrative rights within the site.
