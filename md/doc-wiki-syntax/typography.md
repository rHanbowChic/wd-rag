# Typography

> Documentation > Wiki Syntax > Typography

If you do care about typography there are a few ways to improve it in your text:

| you type | you get |
| --- | --- |
| ``quotation'' | “quotation” |
| `quotation' | ‘quotation’ |
| ,,quotation'' | „quotation” |
| <<quotation>> | «quotation» |
| >>quotation<< | »quotation« |
| dots... | dots… |
| em -- dash | em — dash |

Note: em dash works only when surrounded by spaces.
