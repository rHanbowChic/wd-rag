                    Hints &amp; Tips
                </div>
<div id="page-content">
<ul>
<li>You can use dataform fields to set headings in your page.</li>
<li>The wiki fieldtype is very powerful as it allows the user to use all the formatting and other wiki syntax. It is therefore more useful than the text fieldtype</li>
<li>You can allow the user to set their own CSS for particular elements on the page by passing the contents of a dataform field into a css module.</li>
</ul>
<p>If you want to know how to achieve any of these or other dataform features on your site, please contact us on the community site at <a href="http://community.wikidot.com/forum:start">http://community.wikidot.com/forum:start</a></p>
