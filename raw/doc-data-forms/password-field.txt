
<div class="page-title">
                    The 'password' field type
                </div>
<div id="page-content">
<p>This lets the user enter masked text. To the user, each character they type is replaced by an asterisk ( * ).</p>
<div class="code">
<pre>
<code>[[form]]
fields:
  pass:
    type: password
[[/form]]</code>
</pre></div>
<p><strong>Important:</strong> Entered text is not encrypted, you can always read it in page source.</p>
