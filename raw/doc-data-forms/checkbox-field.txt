
<div class="page-title">
                    The 'checkbox' field type
                </div>
<div id="page-content">
<p>Defines a checkbox field, stored in the form data as 0 or 1. For example:</p>
<div class="code">
<pre>
<code>[[form]]
fields:
  onions:
    label: Do you want onions?
    type: checkbox
  salami:
    label: How about extra salami?
    type: checkbox
    default: 1
[[/form]]</code>
</pre></div>
<p>The specific properties you can use on a checkbox field:</p>
<ul>
<li><strong>default</strong>: defines a default value for the field shown on new pages.</li>
</ul>
