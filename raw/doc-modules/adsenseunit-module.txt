<p>The AdSenseUnit module lets you insert Google AdSense ads into a wiki page. You must have enabled AdSense on your wiki for this module to work. Normally, Wikidot will provide the code for you and you do not need to create your own code to use the AdSenseUnit module.</p>
<p>The syntax for the AdSenseUnit module is:</p>
<div class="code">
<pre>
<code>[[module AdSenseUnit arguments...]]</code>
</pre></div>
<h2 id="toc0"><span>Example</span></h2>
<p>This example shows two left-aligned blocks of adverts:</p>
<div class="code">
<pre>
<code>[[div style="margin-left:auto; margin-right:auto; padding: 10px; margin:0 0em 1em 2em; text-align: center; background-color: transparent; float:left; border: none; width: 15%;"]]
[[module AdSenseUnit label="your add"]]
[[module AdSenseUnit label="your add"]]
[[/div]]</code>
</pre></div>
<h2 id="toc1"><span>Arguments</span></h2>
<p>The <strong>label</strong> argument specifies the label to show, and is mandatory.</p>
