<h2 id="toc0"><span>Description</span></h2>
<p>This module is used to list users watching the page. It will display all users who will receive notification about the changes on this page, so: page watchers, category watchers and site watchers.</p>
<p>Module Watchers also displays "watch/unwatch" options for users.</p>
<h2 id="toc1"><span>Attributes</span></h2>
<table class="wiki-content-table">
<tr>
<th>attribute</th>
<th>required</th>
<th>allowed values</th>
<th>default</th>
<th>description</th>
</tr>
<tr>
<td>noActions</td>
<td>no</td>
<td>"true"</td>
<td>none</td>
<td>does not show "watch/unwatch" options</td>
</tr>
</table>
