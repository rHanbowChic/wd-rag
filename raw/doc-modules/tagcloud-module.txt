<h2 id="toc0"><span>Description</span></h2>
<p>This module creates a customizable <em>tag cloud</em> for tagged pages. Tag Cloud module can be used both for creating plain tag cloud or 3D tag cloud.</p>

<h2 id="toc2"><span>Notes</span></h2>
<p>If you want to change font sizes, both <tt>maxFontSize</tt> and <tt>minFontSize</tt> must be defined and use the same size units.</p>
<p>If you want to change tag colors, both <tt>maxColor</tt> and <tt>minColor</tt> must be defined.</p>
<p>If you choose a custom target page make sure a module <tt>PagesByTag</tt> is there - otherwise there would be no point in linking the tags.</p>
<p>By default tags starting with the underscore "_" are hidden and are not shown by default. Use <tt>showHidden="true"</tt> to display them.</p>
<p>The 3D tag representation is based on the <a href="http://wordpress.org/extend/plugins/wp-cumulus/">WP-Cumulus WordPress plugin</a>.</p>
