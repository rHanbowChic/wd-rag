                    Categories Module
                </div>
<div class="breadcrumb">
<a href="/doc:start">Documentation</a> » <a href="/doc-modules:start">Modules</a> » Categories Module
                    </div>
<div id="page-content">
<p>The Categories module is a tool that displays categories and their pages within a site. This tool will eventually be replaced by <a href="/doc:listpages-module">ListPages</a>, which is more general purpose.</p>
<h2 id="toc0"><span>Attributes</span></h2>
<table class="wiki-content-table">
<tr>
<th>attribute</th>
<th>required</th>
<th>allowed values</th>
<th>default</th>
<th>description</th>
</tr>
<tr>
<td>includeHidden</td>
<td>no</td>
<td><tt>true</tt></td>
<td><tt>false</tt></td>
<td>includes "hidden" (starting with underscore) pages in the listing</td>
</tr>
</table>
<h2 id="toc1"><span>Example</span></h2>
<p>To use Categories, add this code to your page:</p>
<div class="code">
<pre>
<code>[[module Categories]]</code>
</pre></div>
