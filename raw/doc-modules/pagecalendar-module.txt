<h2 id="toc0"><span>Description</span></h2>
<p>The PageCalendar module creates a blogger-friendly calendar that displays the number of pages (articles) created per year and month. It also works great with the <a href="/doc:listpages-module">ListPages module</a> as shown below.</p>
<div class="wiki-note">
<p>We are still working on this module so the final syntax and specifications might change.</p>
</div>
<h2 id="toc1"><span>Attributes</span></h2>
<table class="wiki-content-table">
<tr>
<th>attribute</th>
<th>required</th>
<th>allowed values</th>
<th>default</th>
<th>description</th>
</tr>
<tr>
<td><tt>category</tt></td>
<td>no</td>
<td>comma- or space-separated category names or "*" for whole wiki, or @URL</td>
<td>current category</td>
<td>sets the scope of processed pages</td>
</tr>
<tr>
<td><tt>tags</tt></td>
<td>no</td>
<td>comma- or space-separated tag names with <tt>+</tt> and <tt>-</tt> modifiers,<br/>
or <tt>@URL</tt></td>
<td>none</td>
<td>lists tags that are used as a criteria to select pages, the "+" before the tag makes it required, "-" means "without a tag" and tags without modifiers translate to "pages that have any of those tags";<br/>
"@URL" takes the tags from the URL</td>
</tr>
<tr>
<td><tt>startPage</tt> or <tt>targetPage</tt></td>
<td>no</td>
<td>any valid wiki page</td>
<td>current page</td>
<td>sets the page that will be displayed when any of the dates is clicked</td>
</tr>
<tr>
<td><tt>urlAttrPrefix</tt></td>
<td>no</td>
<td>any alphanumeric</td>
<td>none</td>
<td>prefix for the parameters passed via the URL e.g. to the <a href="/doc:listpages-module">ListPages module</a></td>
</tr>
</table>
<p>If you are using PageCalendar with ListPages, make sure that the <tt>urlAttrPrefix</tt> has the same value in both modules.</p>
<p>Parameters that accept the <tt>@URL</tt> value, i.e. allow for passing the value in the URL, also allow for default values similar to the <a href="/doc:listpages-module#toc8">ListPages module</a>.</p>
<h2 id="toc2"><span>Examples</span></h2>
<p>List pages in the documentation section grouped by year and month, linked with the ListPages module:</p>
<div class="code">
<pre>
<code>[[module PageCalendar category="doc"]]
[[module ListPages category="doc" perPage="7" date="@URL" separate="false" prependLine="||~ Page||~ Date created||~ Created by ||"]]
|| %%linked_title%% || %%date%% || %%author%% ||
[[/module]]</code>
</pre></div>
