<div id="page-content">
<h2 id="toc0"><span>Description</span></h2>
<p>Lists recent changes made to the site content. Supported types of changes:</p>
<ul>
<li>new pages</li>
<li>page content changes (edit source/title)</li>
<li>page rename/move</li>
<li>metadata change (parent page)</li>
<li>attached file actions</li>
</ul>
<p>Changes do not include any forum activity. Use <a href="/doc:recentposts-module">RecentPosts</a> module for this.</p>
<p>The module shows the category name for pages not in default category</p>
<h2 id="toc1"><span>Attributes</span></h2>
<p>No attributes required</p>
<h2 id="toc2"><span>Examples</span></h2>
<div class="code">
<pre>
<code>[[module SiteChanges]]</code>
</pre></div>
