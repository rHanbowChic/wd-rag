                    Code Blocks
                </div>

<div id="page-content">
<p>Create code blocks by using <tt>[[code]]…[[/code]]</tt> tags (each on its own line).</p>
<div class="code">
<pre>
<code>This is an example code block!</code>
</pre></div>
<p>All wiki syntax inside a code block <em>except</em> <span style="white-space: pre-wrap;">[[include]]</span> tags is treated as literal text and not processed. To prevent an include tag from being processed, put a single space in front of it.</p>
<p>Each code block on a page has a unique URL that lets you access it individually. This is especially useful for code blocks that contain CSS code (type = "css"):</p>
<div class="code">
<pre>
<code>http://mysite.wikidot.com/category:page/code
http://mysite.wikidot.com/category:page/code/2  -- second block</code>
</pre></div>
<p>This way you can extract code blocks defined in the page source itself, without taking any _template into account. To access code blocks form page source combined with _template, use the following URLs:</p>
<div class="code">
<pre>
<code>http://mysite.wikidot.com/category:page/code_  -- note the trailing underscore
http://mysite.wikidot.com/category:page/code_/2  -- second block</code>
</pre></div>
<p>To create PHP blocks that get automatically colorized when you use PHP tags, simply surround the code with <tt>[[code <em>type="php"</em>]]…[[/code]]</tt> tags).</p>
<p>To get PHP code colorized you should surround it with &lt;?php.. ?&gt;.</p>
<p>Wikidot.com uses PEAR::Text_Highlighter and supports a number of color schemes. Here is what is supported (allowed type values):</p>
<p>php, html, cpp, css, diff, dtd, java, javascript, perl, python, ruby, xml.</p>
<div style="float: left; width: 45%; margin: 0 2%;">
<div class="code">
<pre>
<code>[[code type="php"]]
&lt;?php
/* comment */
for($i=0; $i&lt;100; $i++){
echo "number".$i."\n";
}
?&gt;
[[/code]]</code>
</pre></div>
</div>
