<div id="page-content">
<p>A comment is a source block that is not rendered in the compiled version. To add a comment to the source use <tt><span style="white-space: pre-wrap;">[!-- ... --]</span></tt> construct, e.g.:</p>
<div class="code">
<pre>
<code>[!--
This text will not be rendered.
--]</code>
</pre></div>
