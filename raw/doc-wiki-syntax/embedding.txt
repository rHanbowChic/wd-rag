<h2 id="toc0"><span><a name="video"></a>Video</span></h2>
<p>To embed video directly into the page use <tt><span style="white-space: pre-wrap;">[[embedvideo]] ... [[/embedvideo]]</span></tt> block.<br/>
Between the the "embedding html code" should be placed from any of the following video online galleries:</p>
<ul>
<li><a href="http://video.google.com">Google Video</a></li>
<li><a href="http://www.youtube.com">YouTube</a></li>
</ul>
<p>E.g.</p>
<div class="code">
<pre>
<code>[[embedvideo]]
&lt;embed style="width:400px; height:326px;" id="VideoPlayback" align="middle"
type="application/x-shockwave-flash"
src="http://video.google.com/googleplayer.swf?docId=263244138622602613"
allowScriptAccess="sameDomain" quality="best" bgcolor="#ffffff" scale="noScale" salign="TL"
FlashVars="playerMode=embedded"&gt; &lt;/embed&gt;
[[/embedvideo]]</code>
</pre></div>
<h2 id="toc1"><span><a name="audio"></a>Audio</span></h2>
<p>To embed audio use <tt><span style="white-space: pre-wrap;">[[embedaudio]] ... [[/embedaudio]]</span></tt> and the html embedding code found on the audio hosting website.</p>
<p>Currently supported are:</p>
<ul>
<li><a href="http://www.odeo.com">Odeo.com</a></li>
</ul>
<p>E.g.</p>
<div class="code">
<pre>
<code>[[embedaudio]]
&lt;embed src="http://www.odeo.com/flash/audio_player_standard_gray.swf"
quality="high" width="300" height="52" name="audio_player_standard_gray"
align="middle" allowScriptAccess="always" wmode="transparent"
type="application/x-shockwave-flash" flashvars="audio_id=99133&amp;audio_duration=282.0&amp;valid_sample_rate=true&amp;external_url=http://www.vitalpodcasts.com/FileSystem/podcasts/89/opensource_episode1.mp3" pluginspage="http://www.macromedia.com/go/getflashplayer" /&gt;
&lt;/embed&gt;&lt;br /&gt;&lt;a style="font-size: 9px; padding-left: 110px; color: #f39;
letter-spacing: -1px; text-decoration: none" href="http://odeo.com/audio/99133/view"&gt;
powered by &lt;strong&gt;ODEO&lt;/strong&gt;&lt;/a&gt;
[[/embedaudio]]</code>
</pre></div>
