<p>Adding underscore to <strong>span</strong> element <strong><span style="white-space: pre-wrap;">[[span_ ]]</span></strong> will truncate whitespaces around it which prevents creation of random <a href="/doc-wiki-syntax:paragraphs-and-newline">new lines and paragraphs</a>. It's simplifices creation of complex HTML syntax like <a href="http://getbootstrap.com/components/">Bootstrap components</a></p>
<div class="alert alert-info">
<p>You can use user-defined <tt>ID</tt> arguments in <strong><span style="white-space: pre-wrap;">[[span]]...[[/span]]</span></strong> tags, which is extremely useful building sites using <a href="http://getbootstrap.com">Bootstrap</a>. Please note that every user-defined <tt>ID</tt> will have a <tt>"u-"</tt> prefix added in the output HTML for the security reasons.</p>
<p>To make your source more readable, you can add the <tt>"u-"</tt> prefix yourself. For example, these 2 bits of wiki syntax will output the same HTML:</p>
<hr/>
<p><strong><tt>"u-"</tt> prefix will be added to <tt>mySpan</tt> automatically when the page is saved</strong></p>
<div class="code">
<pre>
<code>[[span id="mySpan"]]My span element[[/span]]</code>
</pre></div>
<br/>
<strong><tt>"u-"</tt> prefix will not be added to since it already exists</strong>
<div class="code">
<pre>
<code>[[span id="u-mySpan"]]My span element[[/span]]</code>
</pre></div>
<br/>
<strong>HTML output from both examples</strong>
<div class="code">
<pre>
<code>&lt;span id="u-mySpan"&gt;My span element&lt;/span&gt;</code>
</pre></div>
